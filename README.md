# Ball Sort Puzzle Game

A simple terminal-based ball sorting puzzle game written in C.

## How to Play
- Each tube holds up to **4 balls**.
- You can only stack balls of the **same color**.
- Win by sorting all colors into separate tubes.

## Notes
- Works on Windows, Linux, and macOS.

- On Windows, cls is used to clear the screen; on Unix-like systems, clear is used.

- Set the number of tubes (5 to 10) when prompted to control difficulty.

Enjoy the game!
